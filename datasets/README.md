# Datasets

This directory contains sample datasets used across various practicals and projects.

## Dataset Categories

### Standard Datasets
Common datasets for learning and practice:
- CSV files
- Excel files
- JSON data
- Database dumps

### Custom Datasets
Project-specific datasets collected or generated for specific practicals.

## Usage

Datasets are referenced by practicals and projects. Each dataset should have:
- Clear naming convention
- Documentation of columns/fields
- Source attribution (if applicable)

## Popular Public Datasets

You can also use datasets from:
- [Kaggle](https://www.kaggle.com/datasets)
- [UCI Machine Learning Repository](https://archive.ics.uci.edu/ml/index.php)
- [Data.gov](https://www.data.gov/)
- [Google Dataset Search](https://datasetsearch.research.google.com/)

## Note

Large datasets (>10MB) should not be committed to the repository. Use Git LFS or provide download links instead.
