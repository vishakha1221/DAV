# Practicals

This directory contains individual practical exercises for the Data Visualization & Analytics course.

## Structure

Each practical should be organized in its own subdirectory:

```
practical_XX/
├── README.md           # Problem statement and instructions
├── data/              # Dataset(s) for this practical (if any)
├── solution.py        # Python solution file
└── report.pdf         # Report or documentation (if required)
```

## List of Practicals

Practicals will be added as the course progresses.

### Example Topics
- Practical 1: Data Loading and Basic Statistics
- Practical 2: Data Cleaning and Preprocessing
- Practical 3: Exploratory Data Analysis
- Practical 4: Data Visualization with Matplotlib
- Practical 5: Advanced Visualization with Seaborn
- Practical 6: Interactive Plots with Plotly
- And more...

## How to Run

1. Navigate to the specific practical directory
2. Read the README.md for instructions
3. Run the Python script: `python solution.py`
4. Or open in Jupyter: `jupyter notebook solution.ipynb`
